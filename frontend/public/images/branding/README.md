# Branding Assets

This directory contains the Adify logo assets with neon arcade aesthetic.

## Required Files

The following logo files should be placed in this directory:

1. **favicon.png** (192x192px)
   - Play button icon only
   - Used for browser favicon
   - Transparent background with neon green/yellow gradient circle and pink/purple gradient play button

2. **logo-icon.png** (512x512px)
   - Play button icon only
   - Used for app icons and loading states
   - Transparent background with neon green/yellow gradient circle and pink/purple gradient play button

3. **logo-full.png** (1200x400px)
   - Full logo with "Adify" text
   - Used for headers and hero sections
   - Transparent background with neon green/yellow gradient circle and pink/purple gradient play button

## Design Specifications

- **Background**: Transparent (works best on dark backgrounds)
- **Circle Gradient**: Neon green/yellow (#D4E800 → #B8E800 → #88E800 → #6FE89A)
- **Play Button Gradient**: Pink/purple (#8B00FF → #E600FF → #FF00B8)
- **Aesthetic**: Neon arcade style for gaming/rewards feel

## Usage Guidelines

- Use **logo-full.png** for desktop headers, hero sections, and marketing materials
- Use **logo-icon.png** for mobile headers, app icons, and loading states
- Use **favicon.png** specifically for browser favicons
- Always maintain aspect ratio when scaling
- Logo looks best on dark backgrounds (#000000, #0a0a0a, #1a1a1a)
- Consider adding subtle glow effects to enhance the neon aesthetic
